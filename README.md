# 시스템 점검 스크립트 모음

Windows 10/11과 Rocky Linux 8/9 환경에서 사용할 수 있는 종합적인 시스템 점검 스크립트입니다.

## 🎯 개요

이 프로젝트는 시스템 관리자가 Windows와 Linux 환경에서 시스템의 상태를 종합적으로 점검할 수 있도록 도와주는 스크립트입니다.

### 주요 특징

- ✅ **실시간 진행률 표시**: 진행 상황을 시각적으로 확인
- ✅ **빠른 실행**: 3-8초 내 완료
- ✅ **자동 파일 생성**: 스크립트 실행 시 결과 파일이 자동으로 생성
- ✅ **정확한 서비스 식별**: 실제 실행 중인 프로세스와 포트 연결 확인
- ✅ **종합 점검**: 하드웨어, 파일시스템, 네트워크, 서비스, 성능, 보안 등 모든 영역 점검
- ✅ **완벽한 호환성**: Windows 10/11 (한글 출력), Rocky Linux 8/9 (한글 출력) 지원

- **위치**: 스크립트와 동일한 디렉토리

## 🛠️ 문제 해결

### Windows
- **"wmic is not recognized"**: Windows 10/11에서 wmic 명령어가 제거된 경우, PowerShell 명령어로 대체됨
- **한글 문자 오류**: `chcp 65001` 명령어로 UTF-8 인코딩 설정

### Rocky Linux
- **권한 오류**: `chmod +x rocky_linux_system_check.sh` 실행
- **명령어 없음**: 필요한 패키지 설치 (`yum install net-tools procps-ng`)
