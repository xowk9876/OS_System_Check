# 시스템 점검 스크립트 모음

Windows 10/11과 Rocky Linux 8/9 환경에서 사용할 수 있는 종합적인 시스템 점검 스크립트입니다.

## 🎯 개요

이 프로젝트는 시스템 관리자가 Windows와 Linux 환경에서 시스템의 상태를 종합적으로 점검할 수 있도록 도와주는 스크립트입니다.

### 주요 특징

- ✅ **실시간 진행률 표시**: 진행 상황을 시각적으로 확인
- ✅ **빠른 실행**: 3-8초 내 완료
- ✅ **자동 파일 생성**: 스크립트 실행 시 결과 파일이 자동으로 생성
- ✅ **정확한 서비스 식별**: 실제 실행 중인 프로세스와 포트 연결 확인
- ✅ **종합 점검**: 하드웨어, 파일시스템, 네트워크, 서비스, 성능, 보안 등 모든 영역 점검
- ✅ **호환성**: Windows 10/11, Rocky Linux 8/9 지원

## 🚀 사용법

### Windows 10/11

#### 기본 실행 (진행률 표시)
```cmd
windows_system_check.bat
```

#### 상세 정보와 함께 실행
```cmd
windows_system_check.bat --verbose
```

#### 도움말 보기
```cmd
windows_system_check.bat --help
```

### Rocky Linux 8/9

#### 기본 실행 (진행률 표시)
```bash
chmod +x rocky_linux_system_check.sh
./rocky_linux_system_check.sh
```

#### 상세 정보와 함께 실행
```bash
./rocky_linux_system_check.sh --verbose
```

#### 도움말 보기
```bash
./rocky_linux_system_check.sh --help
```

### 결과 파일

- **자동 생성**: 스크립트 실행 시 자동으로 결과 파일이 생성됩니다
- **저장 위치**: 스크립트와 같은 폴더에 저장됩니다
- **파일명 형식**: `system_check_YYYY-MM-DD_HH-MM-SS.txt`

## 📊 점검 항목

### 공통 점검 항목

| 항목 | 설명 |
|------|------|
| **시스템 정보** | OS 버전, 호스트명, 하드웨어 정보 |
| **하드웨어 상태** | CPU, 메모리, 디스크, 온도 |
| **파일시스템** | 사용률, 오류 검사, 마운트 상태 |
| **네트워크** | 인터페이스, 연결, DNS, 라우팅 |
| **서비스/포트** | 열린 포트(실제 프로세스명 포함), 서비스 상태, 방화벽 |
| **성능 모니터링** | 업타임, 로드, 프로세스 |
| **보안** | 사용자, SSH, 업데이트, 로그 |

### 경고 임계값

| 항목 | 경고 기준 | 설명 |
|------|-----------|------|
| 메모리 사용률 | > 90% | 메모리 부족 위험 |
| 디스크 사용률 | > 85% | 디스크 공간 부족 위험 |
| CPU 사용률 | > 80% | CPU 과부하 위험 |

### 자동화 설정

#### Windows 작업 스케줄러
```cmd
# 매일 오전 9시에 실행
schtasks /create /tn "System Check" /tr "C:\path\to\windows_system_check.bat" /sc daily /st 09:00
```

#### Linux Cron 작업
```bash
# 매일 오전 9시에 실행
0 9 * * * /path/to/rocky_linux_system_check.sh
```

## ⚠️ 주의사항

### 권한 요구사항

#### Windows
- **일반 사용자**: 기본적인 시스템 정보만 수집 가능
- **관리자 권한**: 디스크 검사, 서비스 관리, 이벤트 로그 등 모든 기능 사용 가능

#### Rocky Linux
- **일반 사용자**: 기본적인 시스템 정보만 수집 가능
- **Root 권한**: 파일시스템 검사, 시스템 로그, 방화벽 설정 등 모든 기능 사용 가능

### 성능 영향

| 버전 | 실행 시간 | 특징 | 권장 용도 |
|------|-----------|------|-----------|
| **최적화 버전** | 3-8초 | 실시간 진행률 표시, 빠른 점검 | 일일 점검 |
| **기본 버전** | 1-3분 | 상세한 점검, 모든 정보 수집 | 주간/월간 점검 |

- **시스템 리소스**: 최소 (읽기 전용 작업)
- **네트워크 트래픽**: 없음 (로컬 정보만 수집)

## 🔧 문제 해결

### 자주 발생하는 문제

#### Windows

1. **"관리자 권한이 필요합니다" 메시지**
   ```
   해결방법: 명령 프롬프트를 관리자 권한으로 실행
   ```

2. **PowerShell 실행 오류**
   ```
   해결방법: PowerShell 실행 정책 확인
   powershell -Command "Get-ExecutionPolicy"
   ```

#### Rocky Linux

1. **"Permission denied" 오류**
   ```bash
   # 해결방법: 실행 권한 부여
   chmod +x rocky_linux_system_check.sh
   ```

2. **"command not found" 오류**
   ```bash
   # 해결방법: 필요한 패키지 설치
   sudo dnf install net-tools lm-sensors
   ```
